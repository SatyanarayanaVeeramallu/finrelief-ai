# FinRelief AI

## 📌 Project Summary

FinRelief AI is an AI-powered debt relief and financial recovery platform that helps users manage loans, analyze financial health, generate negotiation letters, and receive personalized settlement recommendations using Google Gemini AI.

## 🚀 Live Demo

Frontend: https://finrelief-ai.vercel.app/

Backend API: https://finrelief-ai.onrender.com/

## ✨ Features

- Loan Management System
- Financial Health Analysis
- AI Settlement Recommendations
- AI Negotiation Letter Generator
- Recovery Strategy Suggestions
- FastAPI + React Integration
- Google Gemini AI Powered
